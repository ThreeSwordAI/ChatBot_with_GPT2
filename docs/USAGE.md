# Usage Guide

## User Authentication

1. **Sign Up**:
   - Navigate to the sign-up page.
   - Fill in your name, username and password (username must be unique).
   - Submit the form to create your account.

2. **Login**:
   - Enter your username and password.
   - Upon successful login, you will be redirected to the chat list page.

## Chat Features

1. **Chat List**:
   - After logging in, you will see a list of users you have previously interacted with.
   - Click on any user to start or continue a conversation.

2. **Search for Users**:
   - Use the search bar at the top of the chat list to search for other users by name or username.

3. **Sending Messages**:
   - In the chat box, type your message and click "Send."
   - Your message will appear in the chat window along with the timestamp.

4. **GPT-2 Auto-Generated Text**:
   - Click on the "AutoGenerate" button in the chat interface to generate text using GPT-2.
   - You can edit the generated text before sending it.

## Viewing Chat History

- Each chat window will display the full message history with the selected user.
- You can scroll through the history to see older messages.

## Error Handling

- Incorrect login details will prompt an error message.
- Duplicate usernames will not be allowed during sign-up.
